\begin{array}{cc}
	1&1\\
	1&9\\
	1&-5\\
	1&3
\end{array}
\r]$$
\noindent2. Find the best fitting function $f(t) = c_0 + c_1t$ to data points $(1, 10), (9, 0), (-5, 0), (3, 0)$. \\
\\
\textit{Hint:} We wish to find the least-squares solution to 
$$\l[
\begin{array}{cc}
	1&1\\
	1&9\\
	1&-5\\
	1&3
\end{array}
\r]\l[
\begin{array}{c}
	c_0\\
	c_1
\end{array}
\r] = \l[
\begin{array}{c}
	10\\
	0\\
	0\\
	0
\end{array}
\r].$$
Let $b = \l[
\begin{array}{c}
	10\\
	0\\
	0\\
	0
\end{array}
\r]$ and solve the least-squares problem
$$\l[{c_0 \atop c_1}\r] = \argmin_{x\in\R^2} \|Ax-b\|^2.$$
You don't need to multiply out the matrix product.
