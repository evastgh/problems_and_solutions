\subsubsection*{Problem 2}
In Problem 1, we permit different covariance matrix $\boldsymbol\Sigma_k$ for each cluster $k$. Now suppose we impose that every cluster has identity covariance matrix $\v I$. This changes the expected value of the complete-data log likelihood function (from textbook equation (9.40)) to
\begin{equation*}
\mathbb E_{\v Z}[\ln p(\v X, \v Z \vert \boldsymbol\mu, \boldsymbol\pi)] = \SUM{n=1}N\SUM{k=1}K \gamma(z_{nk}) (\ln\pi_k + \ln\mathcal N(\v x_n \vert \mu_k, \v I) ).
\end{equation*}
(Note the difference in notations for the textbook and the lecture notes.) Show that maximizing the above function with regard to $\boldsymbol\mu_k$ ($k = 1, \cdots, K$) is equivalent to minimizing the $\gamma$-weighted least square error function
\begin{equation*}
E(\boldsymbol\mu_1, \cdots, \boldsymbol\mu_K) = \SUM{i=1}N\SUM{k=1}K \gamma(z_{nk}) \|\v x_n - \boldsymbol\mu_k\|^2. \text{(50 points)}
\end{equation*}
